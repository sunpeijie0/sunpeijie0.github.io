Ubuntu下Snort NFQ Inline Mode 结合iptables实现IPS

Snort

Snort 是入侵检测系统，根据自定义规则，及时发现入侵。

iptables

具有拦截的功能

Inline Mode

在 Inline Mode 下，Snort的drop规则才会丢弃数据包

NFQ

NFQ 是 Snort 运行在DAQ下的一种Module，结合 iptables 的QUEUE_NUM。

NFQ && iptables

Snort 检测出入侵，将入侵源放入 Linux 内核 NFQueue 中，iptables 从中取入侵源，若Snort规则是 drop 或 reject 则 drop 掉这个数据包。
