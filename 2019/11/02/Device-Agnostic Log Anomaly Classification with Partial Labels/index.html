Device-Agnostic Log Anomaly Classification with Partial Labels

Paper Notes 2019/11/02

IWQoS 2018

传统的二分类器包括正样本和负样本，PU Learning解决的是在只有部分标记的正样本以及一些未标记的样本的情况下，二分类器的训练问题。

预处理

不同的设备厂商和设备类型有不同格式的日志消息，现有的模板提取方法需要针对每个设备都进行模板提取。本文将单词分成两类，文本和参数，并过滤掉参数，得到与设备无关的日志。

特征向量

使用 TF-IDF 描述模板的权重。

训练和分类

使用 PU-Learning 将单词分类，并使用 SVM 作为多分类器算法，在 SVM 训练后，可以得到每个模板的训练系数向量，通过将每个向量的值排序，得到 top n 个重要的单词，管理员根据这 n 个重要的单词推断出日志所属的类别。
